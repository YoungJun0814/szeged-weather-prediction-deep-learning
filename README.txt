Weather Prediction Deep Learning Project

Submission contents:
- Deep_Learning_Project.ipynb: final project notebook.
- presentation_slide.pdf: final presentation PDF with slide 26 title typo corrected.
- data/meteostat/*.csv: hourly weather datasets used by the notebook.
- checkpoints/*.pt: saved best model checkpoints from the final training run.

Dataset note:
The original project option was the Weather Dataset / Szeged weather prediction task.
For the final implementation, we used Open-Meteo hourly weather data for six nearby locations and stored the exported CSV files under data/meteostat.


AI tools disclosure:
Generative AI tools were used as support for debugging, checking explanations, improving presentation wording, and preparing final submission materials. The project team reviewed and understood the code, metrics, and conclusions.
